# pycryonium
Манго для режима Winter Zombies Mode в Anime Vanguards.

## Что это?
Здесь содержатся только обновленные файлы, авто-запуск для зимнего ивента.

### Можно/нужно настраивать?
Да, обязательно. Как минимум нужно настроить Roblox-клиент, об этом скоро будет видео на русском, а пока посмотрите у автора оригинала:
- https://youtu.be/kxoBF3VH0mc?si=Ek0SEAG8sa6LpNKG

### Запуск через CMD
Пример:
```powershell
 "D:\documents\GitHub\Winter-Normal-Macro\Python\python3.13.11\python.exe" "D:\documents\GitHub\Winter-Normal-Macro\main.py"
```


### Отличия от оригинального репо:
- Авто-апдейт манго из репозитория оригинального автора и его запуск одним файлом.
- Подмена настроек из оригинального `Winter_Event.py` на настройки из `Settings.json`.
- Фикс проблемы из `Tools\avMethods.py`
- Частично пофикшены проблемы с оригинальным кодом с источника `Mega`
- ПОЛНОЕ отключение макроса при нажатии спец. клавиши

## Остались проблемы с Tesseract/Python?

### Проблемы с Tesseract
Если Tesseract перестает работать или макрос не может обнаружить текст:

- Заново скачайте файлы и разархивируйте их, **или**
- Установите чистую версию Tesseract отсюда:
  [https://github.com/tesseract-ocr/tesseract/releases/tag/5.5.0](https://github.com/tesseract-ocr/tesseract/releases/tag/5.5.0)
- Очистите (удалите) старую папку Tesseract перед установкой.
- По желанию добавьте **директорию Tesseract** (папку, а не сам файл .exe) в системную переменную PATH.

### Проблемы с Python
Если Python не запускается или отсутствуют пакеты:
- Установите чистую версию Python **3.13.11**.
- Установите необходимые пакеты, перечисленные здесь:
  [https://pastebin.com/mS2xFd3m](https://pastebin.com/mS2xFd3m)
